# Blood Moon Horde

A Minecraft Fabric mod that adds a terrifying Blood Moon event every 7 in-game days.

## Features

- **Blood Moon Event**: Every 7 days, a Blood Moon rises with dramatically increased danger
- **10x Mob Spawns**: Hostile mobs spawn at 10 times the normal rate during the event
- **Enhanced Mobs**: All hostile mobs gain Speed II, Strength II, and a glowing effect
- **Blood Moon Champion**: A powerful boss zombie with 100 HP spawns at midnight
- **Warning System**: Players receive a warning message at dusk when a Blood Moon is approaching
- **Survival Rewards**: Players who survive the night receive a Blood Moon Trophy and rare loot (diamonds, emeralds, golden apples, or enchanted golden apples)
- **Night Vision**: Players get night vision during the event to see the chaos unfold

## Requirements

- Minecraft 1.21.1
- Fabric Loader 0.16+
- Fabric API

## Installation

1. Install Fabric for Minecraft 1.21.1
2. Download the JAR from `build/libs/`
3. Place it in your `.minecraft/mods/` folder
4. Make sure Fabric API is also installed

## Building

```bash
gradle build
```

The compiled JAR will be in `build/libs/`.

## Gameplay Tips

- Stock up on weapons, armor, and food before day 7
- Build a secure shelter with good lighting
- The Blood Moon Champion is extremely tough - bring your best gear
- Surviving the night is worth it for the rare loot rewards!
